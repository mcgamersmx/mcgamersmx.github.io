Survival Games

Survival Games es un plugin que tiene como objetivo crear una experiencia totalmente automatizada al estilo los juegos del hambre. El plug-in se ha creado para ser fácil de usar, y para configurarse fácilmente en cualquier tipo de servidor.

¿Qué permite?

  1. - Automatización completa.
  2. - Regeneración automática de Arena.
  3. - Reinicio automatico de cofres.
  4. - Inicio automatico del juego.
  5. - Espectadores.
  6. - Lobby con estadisticas en tiempo real del jugador y de las arenas.
  7. - Multiples arenas.
  8. - Partidas simultáneas.
  9. - Estadísticas basadas en Web.
  10. - Arena fácil de configurar.
  11. - Permisos por Arena.
  12. - Rayos y truenos cuando los jugadores mueren.
  13. - Soporte Multimundos.
  14. - Eventos API - Economía, kits + Más!.

    15. !Además de muchas, muchas más características que incluye este plugin!


    Requisitos:

    Antes de instalar este plugin, se requieren algunas cosas.

  15. - WorldEdit - Necesaria para crear escenarios.

  16. - MySQL base de datos ( opcional ) - Si desea utilizar las estadísticas, usted debe tener una base de datos SQL. -

    17. Descargar:

  17. - Descargar Survival Games [Todas las Versiones] -
  18. - Descargar Survival Games CB 1.6.2-R0.1-
  19. - Descargar Survival Games V1.3.1 -

Los nuevos permisos::

Si usted tiene, por ejemplo, 6 escenarios, y desea que cada jugador sea capaz de unirse a todas las arenas, dar a cada rango los siguientes permisos:

  1. - sg.arena.join. (reemplace con el número de la arena.)
  2. - sg.arena.join
  3. - sg.player.vote
  4. - sg.lobby.join
